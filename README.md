# Wallet-generation-Guide
Wallet generation Guide
````bash
curl -fsSL https://bun.sh/install | bash
source ~/.bashrc
bun --version
````

 Step 2:
````bash
 bun install
````
Step 3:
````bash
bun run build
````

Step 4:
```bash
bun start
```

click the “PORTS” tab open link under forwarded address in browser
